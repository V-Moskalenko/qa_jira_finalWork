# Автотестирование Jira с использованием junit, maven, selenide, allure
Данный комплекс тестов предназначен для edujira.
Тест написан на Java 8

### Какие тесты проводит?
1. Тестирование авторизации
2. Тестирование заведённых задач
3. Тестирование статуса и версии задачи
4. Тестирование заведения новой задачи

## Как запустить?
1. Сохраните данный репозиторий в свою папку.
2. Запустите тест с консоли, использовав команду:
   ```
    mvn clean test
   ```

3. Либо, запустите тест, с помощью класса TestJira, по пути:
   ```
    src\test\java\TestJira
   ```
4. Далее сформируйте отчёт Allure, с помощбю команды:
   ```
    mvn allure:serve
   ```

### Входные данные
Задаются в файле properties:
   ```java
    src\test\resources\props.properties
   ```
